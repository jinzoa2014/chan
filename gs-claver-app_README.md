# GS Claver — Community Web App

An installable, app-like website for a church community, built as a progressive web app (PWA) with a real-time backend.

**Live:** https://gsclaver.online

## What It Is

A mobile-first community web app that behaves like a native app — installable to the home screen, fast, and backed by a real-time database.

## What I Built

- **App-like PWA experience** — installable, works smoothly on mobile, no app store needed.
- **Real-time data** via Firebase, with SQL-backed storage.
- **Built with a modern stack** for fast load and a responsive interface.

## Tech Stack

- **Frontend:** React (Vite)
- **Database:** Firebase (real-time) + SQL
- **Delivery:** PWA (installable)
