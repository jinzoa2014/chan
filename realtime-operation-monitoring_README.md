# Real-Time Operation Monitoring

A live operations monitoring dashboard with alarm annotations for immediate reminders and adjustments, built as an installable progressive web app (PWA).

## The Problem

Operations needed live visibility into process data, plus a way to flag conditions that required immediate attention or adjustment — something static reports and spreadsheets couldn't provide.

## What I Built

- **Real-time monitoring dashboard** displaying live operation data.
- **Alarm annotations** that act as immediate reminders, prompting operators to make adjustments when conditions require it.
- **Progressive Web App (PWA)** — installable for an app-like experience, usable directly from the browser.

## Tech Stack

- **Backend / data:** Python, MongoDB
- **Frontend:** HTML5, vanilla JavaScript
- **Delivery:** PWA (installable)

## Outcome

Operators get live data and timely alarm reminders in one installable app, supporting faster reactions and adjustments on the floor.
